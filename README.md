# E-Commerce Sales Analytics

## Project Overview
Performed end-to-end sales analysis using Python and SQL on the Superstore retail sales dataset.

Dataset: Superstore Retail Sales Dataset

Records:
- 9,994 Orders
- 4 Years of Sales Data (2018–2021)
- Returns Data Included

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- SQL
- MySQL

## Key Analysis
- Sales Analysis
- Profit Analysis
- Customer Analysis
- Shipping Analysis
- Return Analysis
- Monthly Sales Trend

## Key Insights
- Technology category generated the highest sales and profit.
- West region recorded the highest return rate (~11.7%).
- Consumer segment contributed the largest share of overall profit.
- Sales showed a strong upward trend from 2018 to 2021.
